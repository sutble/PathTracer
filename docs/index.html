In the last project we created a mesh network that represented the geometry of an object. In ray generation and primitive intersections we use the concept of rays to figure out how that object would look to the human eye (with considerations of light sources,shadows, mirrors, opaque objects, etc).

Part 1: Ray Generation and Intersection

A ray is determined by its origin point, a direction vector, and a min/max it can occupy. A ray intersects a triangle if for some t along the direction vector (within the min/max t) it is contained within the 3 points defining a triangle. We can formalize this by using barycentric coordinates u,v,w. If a point lies within a triangle defined by p0,p1,and p2 then its barycentric coordinates will follow u+v+w = 1 and none are nonnegative. Since w can be rewritten in terms of u and v (u+v+w = 1), we have 3 unknowns: u,v, and t. We can then use the definition of a ray to formulate a single equation combining these three unknowns and use Cramers rule to find out their values. If t is within min/max t and u,v,w are all nonnegative and add up to 1, then a ray intersects the triangle and the point of intersection is defined as r.o + t*r.d.
